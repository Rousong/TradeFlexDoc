新手入门

开始使用 TradeFlex 的基础指南

TradeFlex 是一款专业的交易记录和投资组合管理应用，帮助投资者：

  • 记录和跟踪所有交易活动
  • 分析投资组合表现
  • 生成详细的统计报告
  • 管理多个投资账户
  • 跨设备同步数据

无论您是新手投资者还是经验丰富的交易员，TradeFlex 都能帮助您更好地管理投资。

创建投资组合非常简单：

  1. 打开 TradeFlex 应用
  2. 点击 "+" 或 "创建投资组合" 按钮
  3. 输入投资组合名称（如：股票投资、加密货币等）
  4. 选择基础货币（人民币、美元等）
  5. 设置初始资金（可选）
  6. 点击 "创建" 完成设置

创建后，您就可以开始记录交易了！

记录交易的步骤：

  1. 进入您的投资组合
  2. 点击 "添加交易" 按钮
  3. 选择交易类型（买入/卖出）
  4. 输入股票代码或名称
  5. 填写交易数量和价格
  6. 选择交易日期
  7. 添加手续费（如有）
  8. 保存交易记录

系统会自动计算您的持仓和收益情况。

账户管理

管理您的账户设置和数据

TradeFlex 提供多种数据备份方式：

云端同步（推荐）
  • 注册 TradeFlex 账户
  • 登录后数据自动云端备份
  • 支持多设备同步
本地导出
  • 进入设置 → 数据管理
  • 选择 "导出数据"
  • 选择导出格式（CSV、Excel）
  • 保存到本地存储

要在多设备间同步数据，请：

  1. 确保所有设备都安装了 TradeFlex
  2. 使用同一个账户登录所有设备
  3. 确保网络连接正常
  4. 在设置中启用 "自动同步"

注意：首次同步可能需要几分钟时间，请保持网络连接稳定。

重置密码的步骤：

  1. 在登录页面点击 "忘记密码"
  2. 输入您的注册邮箱
  3. 查收重置密码邮件
  4. 点击邮件中的重置链接
  5. 设置新密码

如果没有收到邮件，请检查垃圾邮件文件夹，或联系客服获取帮助。

技术支持

解决技术问题和故障排除

遇到应用问题时，请尝试以下解决方案：

基础排查
  1. 重启应用
  2. 重启设备
  3. 检查网络连接
  4. 确保应用是最新版本
进阶解决
  • 清理应用缓存
  • 释放设备存储空间
  • 关闭其他后台应用

如问题持续存在，请联系技术支持并提供详细的错误信息。

数据同步问题的解决方法：

检查网络
  • 确保网络连接稳定
  • 尝试切换网络（WiFi/移动数据）
账户状态
  • 确认已登录账户
  • 检查账户是否正常
手动同步
  • 进入设置 → 账户
  • 点击 "立即同步"
  • 等待同步完成

如仍无法解决，请联系客服获取专业帮助。
